# Chinook SQL Study

Projeto de estudo utilizando o banco de dados **Chinook** para prática de consultas em **SQL Server** e análise de dados.

Este repositório faz parte do meu processo de aprendizado em **Análise e Desenvolvimento de Sistemas (ADS)**, com foco em desenvolvimento backend e manipulação de dados utilizando SQL.

---

# Sobre o Projeto

O **Chinook Database** é um banco de dados de exemplo que simula uma **loja digital de músicas**.  
Ele contém informações sobre artistas, álbuns, faixas musicais, clientes e vendas.

A partir desse banco, foram desenvolvidas diversas **consultas SQL para análise de dados**, permitindo explorar conceitos fundamentais da linguagem SQL.

Principais entidades do banco:

- Artists
- Albums
- Tracks
- Customers
- Invoices
- InvoiceLines

# Objetivo do Projeto

O objetivo deste projeto é **praticar consultas SQL aplicadas a análise de dados**, utilizando um banco relacional completo.

Durante o desenvolvimento das consultas foram utilizados conceitos como:

- JOIN entre múltiplas tabelas
- GROUP BY
- Funções de agregação (SUM, COUNT, etc.)
- Ordenação de resultados
- Análise de vendas
- Relacionamentos entre entidades

---

# Consultas Desenvolvidas

As consultas presentes neste projeto exploram diferentes análises sobre o banco de dados Chinook, como:

- Artistas com maior faturamento
- Faturamento por país
- Top clientes por valor gasto
- Faturamento por gênero musical
- Músicas mais vendidas
- Faturamento por ano
- Quantidade de clientes por país
- Quantidade de clientes atendidos por funcionário
