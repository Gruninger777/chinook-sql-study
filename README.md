# Chinook SQL Study

Projeto de estudo utilizando o banco de dados **Chinook** para prática de SQL Server.

## Sobre o projeto

Este repositório contém exercícios e consultas SQL desenvolvidos durante meus estudos em **Análise e Desenvolvimento de Sistemas**.

O banco Chinook simula uma loja digital de músicas, contendo tabelas como:

- Artists
- Albums
- Tracks
- Customers
- Invoices
- InvoiceLines
